File
简述: 描述文件或文件夹的
1. 创建File对象,构造方法的三种方式
    <1> new File(): 一个参数: 文件的全路径
    <2> new File(): 两个参数: 第一个是文件,第二个参数是下一级路径
    <3> new File(): 两个参数: 第一个是文件的父路径,第二个参数是文件的子路径

2. 获取文件的方法
    <1> getAbsolutePath(): 获取文件的绝对路径
        注: 当创建文件的时候没有输入带盘符(根目录),那么它会将文件所在的项目路径加在文件路径的前面作为绝对路径
    <2> getPath():   获取文件的相对路径
        注: 当文件创建的时候输入的路径是什么,那么获取到的路径就是什么,比如: D:\\AAA\\BBB  结果: D:\\AAA\\BBB
    <3> length(): 获取文件的大小,单位是字节
    <4> getName(): 获取文件名

3. 文件的判断方法
    <1> isFile(): 判断指定的路径下是否是文件
    <2> exists(): 判断指定的路径是否存在
    <3> isHidden(): 判断指定的路径下是否是隐藏文件

4. 创建和删除文件
    <1> delete(): 删除指定目录的文件或文件夹
    注: 已经删除的文件,再次删除会返回false
    <2> createNewFile(): 创建指定目录的文件
    注:
    a)已经创建过的文件或已存在的文件再次创建会失败,因为指定路径下的文件不能重名,无法创建
    b)创建的文件与文件夹不能重名,否则会创建失败

5. 创建文件夹
    <1> mkdir(): 创建单级文件夹
    <2> mkdirs(): 创建多级文件夹
    注:
    a) 当文件夹已经创建了,再次创建会创建失败
    b) 当创建的文件夹没有带盘符(根目录)的时候,它会将文件夹创建在当前项目的目录下面
